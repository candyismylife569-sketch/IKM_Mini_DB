# IKM_Mini_DB
Мини-СУБД: Управление записями пользователей

Простая консольная система управления базой данных для хранения информации о пользователях с возможностью сортировки и сохранения данных в файл.
Основные функции
Управление записями
    1) Добавление новых записей
    2) Просмотр всех записей
    3) Удаление записей по индексу
    4) Поиск по любому полю (имя, возраст, зарплата)
    5) Сортировка по любому полю
Работа с файлами
    1) Сохранение всех данных в текстовый файл
    2) Загрузка данных из файла при запуске
    3) Автосохранение в указанный файл
Структура данных
Каждая запись содержит:
    Имя (name) - строковое поле
    Возраст (age) - целое число
    Зарплата (salary) - число с плавающей точкой
 Формат хранения
Данные сохраняются в формате CSV:
text

Иван,30,50000.5
Мария,25,45000.0
Петр,35,60000.0

 Компиляция и запуск
bash

# Компиляция
g++ -std=c++11 IKM.cpp solutions.cpp -o IKM

Использование
После запуска доступно меню:
    Добавить запись - ввод новых данных
    Вывести все записи - просмотр всей базы
    Удалить запись - удаление по индексу
    Поиск по полю - поиск по имени, возрасту или зарплате
    Сортировка по полю - упорядочивание записей
    Сохранить в файл - явное сохранение изменений
    Загрузить из файла - загрузка данных из файла
    Выход - завершение работы
Технические детали
    Язык: C++11
    Хранение: вектор структур в памяти
    Файл по умолчанию: mini_table.txt
Структура проекта
text

Особенности
    При запуске автоматически загружаются данные из файла
    Изменения сохраняются только при явном вызове сохранения
    Поддерживается работа с кириллицей (русская локаль)
    Пустые строки в файле игнорируются при загрузке
